
# Abyss Editor

Development Setup:
1. Download the repo and open it in unity. It may take some time to import the project fully.
2. Once that is done, in the top menu bar, go to Tools -> ThunderKit -> Settings.
3. Go down to Thunderkit Settings and Click Browse under "Locate and Load game files for project". Go to your subnautica.exe as the destination
4. Then Import and accept any changes requested, you may need to restart the editor once or twice
